Clef's toolchain is in active development. These guides describe the current state of the project and will grow into practical, hands-on material — installation, project setup, and first-program walkthroughs — as the compiler and supporting tools reach public readiness.

For now, start with [Getting Started](/docs/guides/getting-started/) for an overview of the project status and the ways you can engage today. If you want to dive deeper, the [language specification](/spec/draft/) and [design rationale](/docs/design/) are the most developed resources available.
